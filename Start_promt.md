# Промпт для Cursor: Налоговый калькулятор «Бизнес Израиль»

## Описание проекта

Создай веб-приложение — налоговый калькулятор для предпринимателей в Израиле. Калькулятор рассчитывает чистый доход после всех налогов и обязательных отчислений, с детальной разбивкой по каждому виду налога. Интерфейс полностью на **русском языке**.

## Стек технологий

- **React 18+** (Vite)
- **TypeScript** (строгая типизация всех налоговых ставок, порогов и формул)
- **Tailwind CSS** (mobile-first, адаптивный дизайн)
- **Recharts** — для визуализации распределения налогов (pie chart / bar chart)
- Без бэкенда — вся логика на клиенте
- Деплой: статический хостинг (Coolify / Vercel / любой)

### 2. НДС (מע"מ)

```typescript
const VAT = {

  exportRate: 0.0,               // ст. 30(א) — שיעור אפס (НДС 0%)

```

#### Правила НДС 0% (מע"מ בשיעור אפס) — ст. 30(א) Закона о НДС:
- **ст. 30(א)(1)** — экспорт товаров (טובין) за границу с рשимоном (רשימון יצוא)
- **ст. 30(א)(2)** — продажа нематериального актива тошав хуц
- **ст. 30(א)(5)** — оказание услуг тошав хуц (тושב חוץ), при соблюдении условий:
  - Клиент — нерезидент Израиля (не имеет бизнеса/деятельности в Израиле)
  - Клиент находится за пределами Израиля
  - Услуга НЕ оказана одновременно и резиденту Израиля
  - Услуга НЕ связана с активом, находящимся в Израиле
- **ст. 30(א)(7)** — оказание услуг за рубежом (עוסק с основным местом деятельности в Израиле)

**Для калькулятора:** Пользователь указывает процент дохода от экспорта (0–100%). На эту долю применяется НДС 0%, на остальную — 18%.

**Важно:** עוסק פטור не взимает НДС вообще (ни 18%, ни 0%), но и не может зачитывать входящий НДС. Кнопка "экспорт" неактивна для פטור.

```typescript
};
```

**Алгоритм расчёта Битуах Леуми (помесячный):**
1. Берём месячный доход (годовой / 12)
2. Если доход < minIncome → считаем как minIncome (3,442 ₪/мес)
3. Если доход > maxIncome → считаем как maxIncome (51,910 ₪/мес)
4. На часть до reducedThreshold (7,703 ₪) → сниженная ставка
5. На часть от reducedThreshold до maxIncome → полная ставка
6. Считаем отдельно ביטוח לאומי и מס בריאות
7. 52% от суммы ביטוח לאומי вычитаем из налогооблагаемого дохода

   - Полная ставка: X ₪ × 12.83% = Y ₪
   - **Итого Битуах Леуми: Z ₪**
   - Из них 52% (= W ₪) вычтены из налогооблагаемого дохода

4. **Мас бриют (מס בריאות)**

6. **Для חברה בע"מ: Корпоративный налог + дивиденды**
   - Прибыль × 23% = мас хеврот
   - Распределяемая часть × 25%/30% = налог на дивиденды

## АЛГОРИТМ РАСЧЁТА (ПОРЯДОК ВЫЧИСЛЕНИЙ)

### Для עוסק פטור / עוסק מורשה (физлица):

```
1. Годовой оборот (הכנסות)
2. Минус признаваемые расходы (הוצאות מוכרות)
   = Доход до вычетов

3. Расчёт Битуах Леуми + Мас Бриют от дохода
   → Сумма БЛ, Сумма МБ

4. Вычитаемая часть БЛ = Сумма БЛ × 52%
   (это уменьшает налогооблагаемый доход)

5. Пенсионный ניכוי = min(доход × nikuyRate, incomeCap × nikuyRate)
   (это тоже уменьшает налогооблагаемый доход)

6. Керен иштальмут ניכוי = min(доход × 4.5%, incomeCap × 4.5%)
   (это тоже уменьшает налогооблагаемый доход)

7. Налогооблагаемый доход = Доход − Вычитаемая_часть_БЛ − Пенсионный_ניכוי − Керен_ניכוי

8. Подоходный налог по ступеням (от налогооблагаемого дохода)

9. + Мас йесеф (3% на доход свыше 721,560 ₪)

10. − Некудот зикуй (total_nekudot × 2,904)

11. − Пенсионный זיכוי = min(доход × 5.5%, incomeCap × 5.5%) × 35%

12. Итого подоходный налог = max(0, результат шагов 8+9−10−11)

13. Чистый доход = Доход − БЛ − МБ − Подоходный_налог − Пенсионные_отчисления − Керен_взнос
```

### Для חברה בע"מ:

```
1. Прибыль компании (оборот − расходы)
2. Корпоративный налог = Прибыль × 23%
3. Прибыль после налога = Прибыль × 77%
4. Дивиденды = Прибыль_после_налога × %_распределения
5. Налог на дивиденды = Дивиденды × 25% (или 30% для бааль манийот маhути)
6. Мас йесеф на дивиденды (если общий доход > 721,560):
   - 3% базовый + 2% капитальный = 5% на дивиденды свыше порога
7. Итого нагрузка на распределённую прибыль:
   - Корпоративный: 23%
   - Дивидендный: 30% от 77% = 23.1%
   - Совокупный: ~46.1% (без мас йесеф)
8. Нераспределённая прибыль остаётся в компании (налог уже 23%)
9. На нераспределённую часть: показать предупреждение о новом налоге
   на нераспределённую прибыль (6% обязательное распределение)
```

## ВАЖНЫЕ EDGE CASES

1. **Осек патур с оборотом выше порога:**
   - Показать предупреждение: «Ваш оборот превышает порог (122,833 ₪). Необходимо зарегистрироваться как осек мурше!»
   - Рассчитать НДС на сумму превышения (18% из своего кармана)

2. **Оле хадаш с нулевым доходом в Израиле:**
   - Некудот всё равно считаются (могут перенестись если есть другой доход)
   - Показать: «Ваши некудот зикуй не использованы и не переносятся на следующий год»

3. **Доход ниже минимального порога БЛ:**
   - Битуах Леуми считается от минимума (3,442 ₪/мес)

4. **Доход выше максимального порога БЛ:**
   - Битуах Леуми считается только до максимума (51,910 ₪/мес)

5. **Мас йесеф при חברה:**
   - Считается от совокупного дохода физлица (зарплата + дивиденды)
   - Если общий > 721,560 → 3% + 2% на капитальную часть

6. **НДС при смешанных доходах (экспорт + внутренний рынок):**
   - Входящий НДС зачитывается ПОЛНОСТЬЮ (и на экспортные, и на внутренние)
   - Фактически при 100% экспорте осек мурше ПОЛУЧАЕТ возврат НДС

7. **Некудот зикуй не могут дать отрицательный налог:**
   - min(налог, зачёт) → налог не ниже 0
